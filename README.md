# Spotify Clone 

## Overview
A responsive Spotify Clone built using HTML, CSS, and JavaScript. This project recreates the look and feel of Spotify's web player with a clean and modern user interface.

## Features
- Responsive Design
- Sidebar Navigation
- Music Player UI
- Sticky Navigation Bar
- Playlist Cards
- Modern Layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
